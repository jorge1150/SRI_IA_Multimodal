# PASOS DE INSTALACIÓN (orden importante):
#
# 1. Crear entorno con Python 3.12:
#      /usr/local/bin/python3.12 -m venv venv
#      source venv/bin/activate
#
# 2. PortAudio (solo macOS, una sola vez):
#      brew install portaudio
#
# 3. Instalar dependencias:
#      pip install -r requirements.txt
#
# 4. Instalar Ollama y modelos LLM:
#      brew install ollama
#      ollama serve           ← en otra terminal
#      ollama pull qwen2.5:3b-instruct-q4_K_M
#      ollama pull moondream
#
# 5. Descargar modelo Piper TTS en español:
#      python audio/download_piper.py
#
# 6. Cargar documentos SRI en data/ y construir base vectorial:
#      (Copiar PDFs/TXTs del SRI en una subcarpeta de data/, ej. data/IVA/ —
#      el nombre de la carpeta se usa como tipo de normativa)
#      python rag/build_db.py
#
# 6b. (Opcional) Construir grafo de conocimiento GraphRAG:
#      python scripts/build_graph.py
#
# 7. Lanzar sistema:
#      python app.py           ← http://localhost:7865
#
# 8. (Opcional, tesis) Benchmark RAG vs GraphRAG vs Híbrido + RAGAS:
#      python scripts/run_benchmark.py --limit 5    ← prueba rápida
#      python scripts/run_benchmark.py              ← corrida completa
